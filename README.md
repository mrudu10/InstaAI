# Reels Clone (Next.js + TypeScript)

Minimal Instagram Reels-like clone built with Next.js + TypeScript. Includes a simple API returning mock video data and a vertical feed with auto-play behavior.

Quick start

```bash
npm install
npm run dev
```

Open http://localhost:3000
